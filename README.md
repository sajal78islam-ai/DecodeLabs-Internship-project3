# Frontend Developer Portfolio

A responsive, accessible, and high-performance single-page portfolio website showcasing web development milestones completed during the DecodeLabs Industrial Training program. 



## 🛠️ Technology Stack

* **Structure:** Semantic HTML5 to uphold modern web document standards.
* **Styling:** Custom CSS3 leveraging independent layout variables, smooth CSS transitions, and flexible responsive media queries.
* **Interactivity:** Modern Vanilla JavaScript (ES6+) strictly utilizing optimized modern DOM manipulation practices and standard browser observer APIs.

## 📦 How to Run Locally

1. Clone this repository to your machine.
2. Ensure the structural directory contains your asset files:
   * `index.html`.
   * `portfolio.css`.
   * `portfolio.js`.
3. Open `index.html` directly inside your favorite web browser or run it with a development tool extension like **Live Server**.
